        Welcome to LedgerX

        Monthly Performance Report Generator

        LedgerX automatically validates your trading data, calculates client performance, generates reporting summaries, detects data issues, and creates a ready-to-use report for GHL.

        How to Generate Monthly Performance Report

        Step 1

        Update the LedgerX Template

        Populate the template using the following sources.

        Client Database

        Source: Account Tracker · Sheet: Account Tracker

        • Copy Columns A to C.
        • Paste as VALUES ONLY.
        • Replace every “N/A” with blank cells.
        • Keep numbers as numbers.
        • Do not modify column structure.

        Raw Data

        Source: STT Dashboard · Navigate: Trades

        • Copy every page.
        • Copy the entire table.
        • Paste as VALUES ONLY into Raw Data.
        • Include every page.
        • Preserve the original format.
        Step 2

        After updating the LedgerX Template, open LedgerX.

        Step 3

        Select the reporting month. The current year is detected automatically — you never choose the year.

        Step 4

        LedgerX automatically:

        • Reads the updated template
        • Validates records
        • Calculates performance
        • Detects errors
        • Generates summaries
        • Creates the Performance Summary spreadsheet
        • Saves it into Google Drive

        No manual processing required.

        Step 5

        When processing finishes, review: Performance Summary and Errors (if any).

        Step 6

        Copy the generated note and paste it directly into GHL. No additional formatting required.

        Need Help?

        Ensure the LedgerX Template has been updated before generating reports. LedgerX only processes the latest available data.
